# Data and Code Availability

The source code, locked configuration files, literature-prior tables,
leave-one-out back-testing scripts and outputs, aggregate local-validation
outputs, docking summary outputs, and figure-generation workflows supporting the
computational results are included in this public repository.

The de-identified local clinical cohort supporting the retrospective validation
has been deposited in the National Genomics Data Center OMIX database under
accession `OMIX017364`. Patient-level clinical feature, outcome, prediction, and
treatment-ranking tables are not redistributed in this GitHub package.

Restricted or credentialed third-party data, including COSMIC, OncoKB, DrugBank,
and copyrighted source PDFs, are not redistributed. Scripts, configuration
files, source-status manifests, and open fallback tables are provided so that
users with appropriate access rights can regenerate those components.
